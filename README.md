## About
Personal dotfiles, almost all of which was assimilated and modified from all over the place.

| Item | Value |
| :--- | :---- |
| OS | Manjaro |
| DE | Xfce |
| WM | Bspwm |
| Bar | Lemonbar |
| Compositor | picom |
| Shell | bash |
| Terminal | xfce4-terminal |
| Font | Inconsolata |
| Mono Font | Fira Code |
| Notification | xfce4-notifyd |
| Launcher | Rofi, dmenu |
| Wallpaper | feh |
| Text Editor | Sublime Text, nano |
